# asset-library

Downloaded, license-clean 3D asset packs for CityBuilder. Every asset here is
**CC0 / CC-BY / CC-BY-SA** (never NC/ND) so it can ship in the product and feed
the generation pipeline — see PRD §6.3 / §7E.

## Packs

### `toxsam-polygonal-mind/` — 991 GLB models · CC0 1.0 (public domain)

- **Source:** [ToxSam/open-source-3D-assets](https://github.com/ToxSam/open-source-3D-assets)
  (registry) → [ToxSam/cc0-models-Polygonal-Mind](https://github.com/ToxSam/cc0-models-Polygonal-Mind) (models).
  Models by **Polygonal Mind**, catalogued by ToxSam.
- **License:** CC0 1.0 Universal — no attribution required (credit appreciated).
- **Format:** GLB (self-contained), with per-model PNG thumbnails.
- **Labels:** already tagged with `Theme` + `Category` in the source registry —
  no manual labelling needed. Captured into `manifest.json`.

**Reality check:** these are *stylized metaverse environment packs* (vaporwave,
Egyptian, medieval, sci-fi), not photoreal real-world city assets. But the
**category labels** surface genuinely usable city content spread across all
collections. Filter by category rather than by collection.

## `manifest.json`

Generated by `build-manifest.mjs` from the source registry (`.registry/`). Per
asset: `id`, `name`, `collection`, `category`, `theme`, `license`,
`cityRelevant`, `localPath`, `sourceUrl`, `thumbnail`, `fileSize`. Regenerate
with `node build-manifest.mjs`.

## Integrating with the existing scanner

CityBuilder's manifest scanner reads `assets/library/<pack>/`. To let it pick
these up, either move `toxsam-polygonal-mind/` under `assets/library/`, or point
the scanner at this folder. The scanner will re-measure bounding boxes, count
triangles, and apply its own poly/size budget flags (these are game/metaverse
assets, so some may exceed the street-furniture triangle budget and get flagged
`high-poly` — that's expected; they stay advisory per §7E).

## Not included (and why)

- **3DStreet** ([3dstreet-assets-source](https://github.com/3DStreet/3dstreet-assets-source)) —
  ideal glTF street content, but **CC-BY-NC-SA 4.0** (NonCommercial) → fails the
  license gate. Would need a commercial license from 3DStreet LLC.
- **CARLA** — assets are CC-BY but distributed as ~25 GB Unreal Engine cooked
  `.uasset` binaries (not glTF), and high-poly. No bulk export path.
- **PLATEAU-GIS-Converter** / **OSM2World** — converter *tools*, not asset
  libraries. (The PLATEAU *dataset* they process is CC-BY realistic city
  buildings, worth a separate look for base massing.)
